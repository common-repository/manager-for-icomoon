Manage icomoon package.

== Description ==
This plugin makes it easy to integrate an icon font built from the Icomoon site. Icons then easily integrate into your content via a shortcode


== Installation ==
- Upload the plugin files to the /wp-content/plugins/managerforicomoon/ directory, or install the plugin through the WordPress plugins screen - directly.
- Activate the plugin through the Plugins screen in WordPress
- Use the Icomoon's settings to configure the plugin
